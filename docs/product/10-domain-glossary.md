# Domain Glossary

## Purpose

Define core product-domain terms used across requirements, data modeling, reporting, and AI features.

## Terms

| Term | Meaning |
| --- | --- |
| Owner | A configurable person/entity used for attribution and filtering |
| Joint | Shared ownership bucket |
| Ownership Allocation | Percentage-based attribution of account/transaction value |
| Retrofit | Applying a configuration change historically |
| Rule | User-defined categorization or characterization logic |
| Recharacterization | Changing the financial interpretation of a transaction |
| Split | Dividing one transaction into multiple classified parts |
