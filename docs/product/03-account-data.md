# Account and Financial Data Requirements

## Account Types

The app must support:

- Checking
- Savings
- Credit cards
- Brokerage
- Retirement
- Mortgage
- Personal loans
- Student loans
- Auto loans
- Manual assets
- Manual liabilities

## Linked Accounts

Linked accounts should support:

- Balance sync
- Transaction sync
- Account metadata
- Institution metadata
- Connection health/status

## Manual Accounts

Manual accounts should allow the user to track:

- Asset name
- Liability name
- Current value
- Starting value
- Valuation date
- Owner allocation
- Notes

## Required Account Metadata

Each account should have:

- Display name
- Institution name
- Account type
- Account subtype
- Ownership configuration
- Active/inactive status
- Include/exclude from net worth
- Include/exclude from budgeting
- Include/exclude from tax planning
